# node-front-template

template for back end

English | [中文](README-CN.md)

## clone

```
$ git clone --bare https://github.com/greedlab/node-back-template.git your-project-name
```

## develop env

```
$ npm install -g gulp nodemon
$ npm install
$ gulp build
$ gulp watch
$ npm run develop
```

## production env

```
$ npm install -g gulp pm2
$ npm install
$ gulp release
$ npm run release
```
